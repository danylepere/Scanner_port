# Scanner_port 🔒

Mon premier Script Python de cybersécurité - Niveau Licence 1

### Description
Outil qui vérifie si un port TCP est ouvert ou fermé sur une adresse IP ou un nom de domaine.
Exemple : Tester si le port 22 SSH ou 80 HTTP répond.

### Comment l'utiliser 
1. Lancer le Script : " python-port.py"
2. Entrer l'IP ou domaine : exemple : 127.0.0.1 ou google.com
3. Entrer le port à tester : exemple : 22 , 80 , 443
 
### Compétences utilisées 
- Module "socket" en python
- Gestion des erreurs avec "try/except"
- Conversion de type "int(input())"
- F-strings pour l'affichage
  
### Ce que j'ai appris
Les bases du réseau + comment python communique avec les ports.
Projet fait pour m'entrainer avant TryHackMe.
